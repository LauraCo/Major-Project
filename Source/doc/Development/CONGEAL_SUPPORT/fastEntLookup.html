fastEntLookup

PURPOSE

This function takes the mean of a set of binary images

SYNOPSIS

function ent=fastEntLookup(meanImg)

DESCRIPTION 

 This function takes the mean of a set of binary images 
 and returns the summed pixelwise entropies. It uses a lookup
 table to get the entropy.
